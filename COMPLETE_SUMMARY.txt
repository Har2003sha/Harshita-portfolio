## 🎨 Easy Customizations

### Change Your Name & Info
Edit `app.py`:
```python
about_info = {
    'name': 'Your Name',
    'title': 'Your Title',
    'bio': 'Your description',
    ...
}
```

### Add Your Projects
Edit `app.py`:
```python
projects = [
    {
        'id': 1,
        'title': 'Your Project',
        'description': 'Description',
        'technologies': ['Tech1', 'Tech2'],
        'image': 'project1.jfif',
        'details': 'Full details'
    },
    ...
]
```

### Change Colors
Edit `static/css/style.css`:
```css
:root {
    --warning-color: #ffc107;  /* Yellow accent */
    --primary-color: #667eea;  /* Purple primary */
    /* ... change colors ... */
}
```

### Update Contact Info
Edit `templates/contact.html`:
- Email
- Phone
- Social links
- Location
